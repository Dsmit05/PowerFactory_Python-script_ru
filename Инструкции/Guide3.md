## Расчет УР и КЗ

Для доступа к объектам в активном исследовании используем команду `GetFromStudyCase()`,
данная функция позволяет запускать различные команды расчета:
```python
UR=app.GetFromStudyCase('ComLdf')
UR.Execute()  # Запуск расчета УР

KZ=app.GetFromStudyCase('ComShc')
KZ.Execute()  # Запуск расчета КЗ
```
##### Основные аргументы для запуска расчетов:
| Атрибуты    | Описание                       | 
| ------------|:------------------------------:| 
|ComLdf       |расчет УР                       |
|ComShc       |расчета КЗ                      |
|ComStatsim   |квазидинамическое моделирование |
|ComSimoutage |анализ аварийного режима        |
|ComRel3      |определение надежности          |
|ComMod       |модальный анализ                |
|ComGenrel    |балансовая надежность           |
|ComNmink     |аварийное событие               |

Есть и другие расчеты, но выполнение тех или иных команд зависит от условий лицензии PF.

Получив один из выше представленных объектов мы можем изменять его параметры расчета, например в расчете КЗ 
для изменения повреждения на 2-х фазное КЗ поменяем параметр `iopt_shc`:
```python
KZ=app.GetFromStudyCase('ComShc')
KZ.iopt_shc = '2psc'
KZ.Execute()
```
Ниже приведены основные параметры для расчета КЗ и их аргументы.

##### Метод расчета(`iopt_mde`):
| Значение    | Описание                         | 
| ------------|:--------------------------------:| 
|0            |VDE 0102 Часть 0 / DIN EN 60909-0 |
|1            |МЭК 60909                         |
|2            |ANSI                              |
|3            |полный                            |
|4            |МЭК 61363                         |
|5            |61660 (DC)                        |
|6            |ANSI/IEEE 946 (DC)                |
|7            |VDE 0102 Part 10 (DC)/DEN EN 61660|

##### Вид повреждения(`iopt_shc`):
| Значение    | Описание                         | 
| ------------|:--------------------------------:| 
|Зрsс         |3-фазное КЗ                       |
|2psc         |2-фазное КЗ                       |
|spgf         |1-фазное К3 на землю              |
|2pgf         |2-фазное КЗ на землю              |
|spnf         |1-фазное К3 на нейтраль           |
|song         |1-фазное К3 на нейтраль и землю   |
|3rst         |3-фазное К3 (несим. )             |


Аргументы для других параметров вы можете увидить при наведении курсора мыши на элемент, или в менеджере данных.
